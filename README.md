## How can I edit this code?

There are several ways of editing your application.

**Use Lovable**

Simply visit the [Lovable Project Page](https://lovable.dev/projects/408c65ac-4283-4e0b-84cc-421977dc8e90) and start prompting.

**Edit and deploy directly from GitHub**

- [Connect your Lovable project to GitHub](https://docs.lovable.dev/tips-tricks/github-integration/)
- Make changes to the code on GitHub
- Changes will be automatically deployed to your Lovable project

**Edit locally**

- [Connect your Lovable project to GitHub](https://docs.lovable.dev/tips-tricks/github-integration/)
- Clone the repository to your local machine:
  ```sh
  git clone <YOUR_GITHUB_REPO_URL>
  ```
- Install dependencies and run the development server:
  ```sh
  cd <YOUR_PROJECT_NAME>
  npm install
  npm run dev
  ```
